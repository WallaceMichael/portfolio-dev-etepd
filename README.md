# 💻 Portfólio Dev - ETE Porto Digital

Projeto desenvolvido durante a oficina prática de **Git e GitHub**, com foco na criação e publicação de um portfólio simples.

---

## 🚀 Sobre o projeto

Este projeto tem como objetivo introduzir conceitos básicos de versionamento de código e presença profissional online, permitindo que os alunos desenvolvam e publiquem seu primeiro portfólio utilizando ferramentas do mercado.

Durante a oficina, os participantes aprendem na prática como criar um repositório, versionar arquivos e disponibilizar seus projetos na internet.

---

## 🎯 Objetivos da oficina

- Apresentar o Git e GitHub de forma prática  
- Ensinar comandos básicos de versionamento  
- Desenvolver um portfólio simples em HTML e CSS  
- Publicar o projeto online  
- Incentivar a criação de projetos próprios  

---

## 🛠️ Tecnologias utilizadas

- HTML5  
- CSS3  
- Git  
- GitHub
